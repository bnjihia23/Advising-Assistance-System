**CS-499 Category 2 Artifact - Advising Assistance**



Author: Brian Njihia



Completed: 23 November 2025



**Overview**



This program reads course data from a CSV file and provides a menu-driven course planner interface that allows users to load the dataset, view all courses sorted alphanumerically by course number, and look up detailed information about a specific course, including its prerequisites.



This folder contains two Visual Studio solutions:

1. AdvisingAssistance - Original version using a linked list
2. AdvisingAssistance_Enhanced - Enhanced version using a binary search tree



To run this project, you will need:

* Visual Studio 2022 (or 2019)
* Windows 10 or later



**Build and Run Instructions**

1. Open the appropriate .sln file in Visual Studio.
2. Build the project: **Build → Build Solution**
3. Run the program: **Debug → Start Without Debugging**



**The name of the .csv file containing the course data is** CS_300_ABCU_Advising_Program_Input.csv. **The CSV file is already included in each project folder and contains the course data.**
